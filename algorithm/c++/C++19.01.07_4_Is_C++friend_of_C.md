1. ## function

   > 구조 역시  `c`와 똑같다.

   - 코드 예시

     ```c++
     #include <iostream>
     using namespace std;
     
     void print_square(int arg);
     int main() {
       int i;
     
       cout << "제곱할 수 ? : ";
       cin >> i;
     
       print_square(i);
     
       return 0;
     }
     
     void print_square(int arg) { cout << "전달된 인자 : " << arg * arg << endl; }
     ```

   - 실행 결과

     ```
     제곱할 수? : 12
     전달된 인자 : 144
     ```

2. ## 레퍼런스의 도입

   > `c` 에서는 `c++`에서 레 퍼런스 개념이 생김. 

   - 코드 예시 1-1

     ```c++
     #include <iostream>
     
     using namespace std;
     int change_val(int *p) {
       *p = 3;
     
       return 0;
     }
     int main() {
       int number = 5;
     
       cout << number << endl;
       change_val(&number);
       cout << number << endl;
     }
     ```

   - 코드 예시 1-2

     ```c++
     #include <iostream>
     
     using namespace std;
     int change_val(int &p) {
       p = 3;
     
       return 0;
     }
     int main() {
       int number = 5;
     
       cout << number << endl;
       change_val(number);
       cout << number << endl;
     }
     ```

     - 코드 1-1 과 달리 코드 1-2에서는 변수 number 앞에 `&` 를 붙이지 않고 함수 정의 부분에서도 `*p`가 아닌  `&p`를 받는다. 
     - 이것을 레퍼런스라고 하며 `p`는 `number`의 다른 이름이 되는 것이다.

     ```c++
     int& ref = number;
     ```
